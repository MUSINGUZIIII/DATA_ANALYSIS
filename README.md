# DATA_ANALYSIS
Data analysis using jupyter and python
The initial titanic dataset was not clean, it contained a lot of missing values.
I used median for missing age values.
I used mode for missing embarked values.
I removed columns such as cabin that were not necessary for survival insights.
I also used Microsoft powerBI to generate insights from the clean dataset.


The iris dataset has 3 different leaf types with different sepal and petal lengths. I used matplotlib and numpy to show correlations between the various sepal and petal size variations.
